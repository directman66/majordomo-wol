Информация
Модуль отправляет магические пакеты Wake On Lan. При нажатии кнопки Discovery производится запрос arp таблицы и запоминаются доступные данные устройств. По кнопке pingall производитяс опрос всей таблицы методом ping. Если в результате запроса никаких данные не появилось, скорее всего у вас не установлен пакет net-tools. Для установки наберибе в консоли linux следующие код: sudo apt install net-tools
Для того, чтобы отправить пакет WOL, нужно нажать на кнопку с mac-адресом устройства.
Вызов из сценариев
include_once(DIR_MODULES . 'wol/wol.class.php');
$wake = new wol();
$wake->WakeOnLan("255.255.255.255", '74:d4:35:e4:82:f7');
$wake->WakeOnLan('192.168.255.255','74:d4:35:e4:82:f7');
$wake->WakeOnLan('192.168.0.255','74:d4:35:e4:82:f7');
$wake->WakeOnLan('192.168.1.255','74:d4:35:e4:82:f7');

Последняя строка выбирается в зависимости от вашей домашней подсети и настроек роутера. Можно пакет отправить по все подсети, ничем это не грозит.
